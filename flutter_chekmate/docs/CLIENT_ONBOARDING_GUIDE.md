## üì± Android APK Installation

For the full native Android experience, install the ChekMate APK on your Android device.

### Requirements

- **Android Version:** Android 7.0 (Nougat) or higher
- **Storage Space:** 100 MB free space
- **Internet Connection:** Required for all features

### Download Link

**Direct Download:** https://github.com/Isakainovorium/ChekMate/releases/download/v1.0.0-beta/app-release.apk

**Or visit the release page:** https://github.com/Isakainovorium/ChekMate/releases/tag/v1.0.0-beta

### Installation Instructions

#### Option 1: Direct Download on Android Device (Easiest)

1. **On your Android device**, open this link in your browser:
   ```
   https://github.com/Isakainovorium/ChekMate/releases/tag/v1.0.0-beta
   ```

2. **Tap "app-release.apk"** (58.76 MB file)
   - Download will start automatically
   - Wait for download to complete

3. **Open the downloaded file**
   - Tap the download notification, OR
   - Go to your **Downloads** folder and tap the APK file

4. **Enable "Install from Unknown Sources"** (if prompted)
   - **Android 8.0+:** Tap **"Settings"** ‚Üí Enable **"Allow from this source"**
   - **Android 7.x:** Go to **Settings** ‚Üí **Security** ‚Üí Enable **"Unknown Sources"**

5. **Tap "Install"**
   - Wait for installation to complete (10-30 seconds)

6. **Tap "Open"** to launch ChekMate!

#### Option 2: Download on Computer, Transfer to Device

1. **On your computer**, download the APK:
   - Visit: https://github.com/Isakainovorium/ChekMate/releases/tag/v1.0.0-beta
   - Click **"app-release.apk"** to download

2. **Connect your Android device** to computer via USB cable

3. **Transfer the APK file**
   - Copy `app-release.apk` to your device's **Downloads** folder

4. **On your device**, open the **Files** or **My Files** app

5. **Navigate to Downloads** folder

6. **Tap the APK file** and follow steps 4-6 from Option 1 above

### Security Note: "Install from Unknown Sources"

When installing apps outside of Google Play Store, Android shows a security warning. **This is normal and expected.**

**Why this happens:**
- Android protects users from installing apps from untrusted sources
- This APK is from GitHub (a trusted platform) but not from Play Store
- The app is **signed with a valid release certificate**
- The app is **100% safe to install**

**After installation:**
- You can disable "Unknown Sources" again for security
- ChekMate will continue to work normally

### Troubleshooting Common Issues

**Problem:** "App not installed" error  
**Solution:** 
- Make sure you have enough storage space (100 MB)
- Try uninstalling any previous version first
- Restart your device and try again

**Problem:** Can't find the downloaded APK  
**Solution:**
- Check your **Downloads** folder
- Check your browser's download history
- Try downloading again

**Problem:** Installation blocked by security settings  
**Solution:**
- Follow the "Enable Unknown Sources" steps above
- Make sure you're allowing installation from your browser or Files app
